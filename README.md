# Memory Game

## Overview
The Memory Game is a fun and interactive game where players flip cards to find matching pairs. This project is designed for 2 to 4 players and includes a simple user interface, card flipping animations, and a scoring system to determine the winner.

## Project Structure
```
memory-game
├── index.html        # Main HTML document for the game interface
├── css
│   └── styles.css    # Styles for the game layout and card designs
├── js
│   └── game.js       # JavaScript logic for game functionality
├── assets
│   └── (placeholders for external image URLs or SVGs) # Images for the cards
├── tests
│   └── game.test.js  # Unit tests for game logic
└── README.md         # Project documentation
```

## How to Play
1. Open `index.html` in a web browser.
2. Enter the number of players (2, 3, or 4) when prompted.
3. Players take turns flipping two cards at a time to find matching pairs.
4. If a player finds a match, they earn a point and get another turn.
5. The game continues until all pairs are found.
6. The player with the most points at the end of the game is declared the winner.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. Ensure that the images for the cards are placed in the `assets` directory.
